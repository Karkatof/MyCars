# MyCars
Приложение для контроля поездок [CoreData] 

### **Оосновные возможности:**
+ Хранение данных на устройстве
+ Сохранение данных о поездках
+ Изменение рейтинга авто

### **Используемые технологии:**
+ Архитектура - MVС
+ Хранение данных - CoreData
+ Верстка - SnapKit
